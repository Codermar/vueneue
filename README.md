**A new start to build SSR (Server-side rendered) applications with Vue CLI 3.x**

## List of plugins

- [@vueneue/ssr](https://github.com/vueneue/ssr): Quick start to add SSR to your application
  ```bash
  vue add @vueneue/ssr
  ```

## Use starter preset

```bash
vue create --preset vueneue/vueneue <appName>
```
